# Azure Assignment Summary

## Region Chosen
I selected UK South as my deployment region because it offers high availability and low latency to my customers mainly located in Africa and Europe. It also has an Availability zone.

## Shared Responsibility Model

In Azure, responsibility is shared between the cloud service provider and the cloud user:

- Microsoft is responsible for:
  - Physical data centers
  - Hardware
  - Network infrastructure

- I am responsible for:
  - Managing my resources
  - Securing my data
  - Configuring access controls

Hence, the security of the cloud is the responsibility of Azure while security in the cloud is that of the user.

This ensures both security and flexibility in cloud usage.
